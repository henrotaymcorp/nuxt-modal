#nuxt-modal

A simple package to manage modals.

## Installation

```shell
yarn add @henrotaymcorp/nuxt-modal
```

Add `@henrotaymcorp/nuxt-modal` to the `modules` section of `nuxt.config.ts`

```typescript
export default defineNuxtConfig({
  modules: [
    '@henrotaymcorp/nuxt-modal'
  ],
  nuxtModal: {
    // @TODO
  }
})
```
